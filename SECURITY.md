# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within DJINN, please do NOT open a public issue.

Instead, please email **bbb.vloger@gmail.com** with standard details. We will address the issue as soon as possible.
