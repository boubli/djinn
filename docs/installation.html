Installation

Follow these steps to get DJINN running on your machine.

Quick Start

Install the DJINN CLI directly from our repository.

git clone https://github.com/boubli/djinn.git
cd djinn
pip install -e .

Requirements

  • Python 3.8+
  • Pip
  • Ollama (recommended for local LLM)

Next Steps

Once installed, verify it works by running:

djinn --help

Then, head over to Configuration to set up your LLM backend.
