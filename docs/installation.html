Installation

Get started with DJINN on Windows, macOS, or Linux.

Prerequisites

  • Python 3.8 or higher
  • Access to an LLM (Ollama running locally, or an API key for OpenAI/DeepSeek)

Install via pip

The recommended way to install is via PyPI (coming soon) or directly from source:

git clone https://github.com/boubli/djinn.git
cd djinn
pip install -e .

Verify Installation

Run the help command to check if everything is working:

djinn --help

Setting up the Backend

By default, DJINN may look for a local Ollama instance. To configure it:

# For Ollama (Local)
djinn config --backend ollama --model llama3

# For OpenAI (Cloud)
export OPENAI_API_KEY="sk-..."
djinn config --backend openai --model gpt-4o
