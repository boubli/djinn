Using the Marketplace

Browse Available Plugins

djinn plugin list              # List installed plugins
djinn plugin browse            # Browse marketplace

Install a Plugin

djinn plugin install <name>
djinn plugin install aws-advanced
djinn plugin install k8s-pro

Uninstall a Plugin

djinn plugin uninstall <name>

Update Plugins

djinn plugin update            # Update all
djinn plugin update <name>     # Update specific

plugin.json Example

  "name": "my-awesome-plugin",
  "version": "1.0.0",
  "description": "My awesome DJINN plugin",
  "author": "yourname",
  "keywords": ["utility", "productivity"],
  "commands": ["my-command-1", "my-command-2"]
}

commands.py Example

from djinn.plugins import Plugin

class MyPlugin(Plugin):
    name = "my-awesome-plugin"
    
    def get_commands(self):
        return {
            "greet": "echo 'Hello from my plugin!'",
            "info": "echo 'Version 1.0.0'"
        }
    
    def generate(self, prompt: str) -> str:
        if "greet" in prompt:
            return "echo 'Hello, World!'"
        return None

Submit to Marketplace

# 1. Create your plugin locally
mkdir my-plugin && cd my-plugin

# 2. Initialize plugin
djinn plugin init

# 3. Test locally
djinn plugin test

# 4. Submit to marketplace
djinn plugin publish

Plugin Guidelines

  • Plugins must be open source
  • Include comprehensive documentation
  • No malicious code or data collection
  • Follow semantic versioning
  • Provide examples and tests
