Configuration

Customize DJINN's behavior, themes, and AI backend.

LLM Backend

DJINN supports multiple backends. Switch between them instantly.

djinn config --backend ollama
djinn config --backend openai
djinn config --backend lmstudio

Choosing Models

Specify which model fits your needs (speed vs precision):

djinn config --model llama3     # Fast, good for daily use
djinn config --model gpt-4      # High precision

Themes

DJINN comes with 5 built-in themes:

Theme Description
default Classic green terminal style
hacker Matrix-inspired aesthetic
ocean Calm blue color palette
purple Vibrant synthwave style
minimal Clean monochrome (High contrast) 
djinn theme purple

View Configuration

To see your current settings:

djinn config --show
